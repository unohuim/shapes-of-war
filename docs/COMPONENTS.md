# Components

This document lists the current game components and resolved action deck composition. It does not define physical production quantities.

## Bases

- Wood Base
- Stone Base
- Metal Base

## Units

- Triangle
- Square
- Circle

Units are represented only by shape.

## Resources

- Wood
- Stone
- Metal

Unit and resource supplies are effectively unlimited for gameplay. Spent resources, discarded units, and Triangles sacrificed for action cards return to circulation.

## Action Cards

The only current action cards are:

- Raid Base
- Resource Theft
- Unit Kill
- Counter

Used action cards go to an action card discard pile after resolution. If the action card deck is empty when a player needs to draw, shuffle the discard pile into a new deck. If both the deck and discard pile are empty, no card is drawn.

## Action Deck Composition

| Card | Count |
| --- | ---: |
| Raid Base | 10 |
| Resource Theft | 10 |
| Unit Kill | 10 |
| Counter | 20 |

## Possible Physical or Digital Representations

The game may later become a card game, a board/card hybrid, or a digital/mobile game. Component representation should remain flexible until the format is chosen.

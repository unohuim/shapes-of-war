# Balance Tables

These tables collect the current numeric foundation. They are not a full balance pass.

## Base Points

| Base | Points | Upgrade Path |
| --- | ---: | --- |
| Wood Base | 3 | Upgrades to Stone Base |
| Stone Base | 5 | Upgrades to Metal Base |
| Metal Base | 7 | Final current base tier |

## Base Upgrade Costs

| From | To | Cost |
| --- | --- | --- |
| Wood Base | Stone Base | 2 Stone |
| Stone Base | Metal Base | 2 Metal |

| Unit | Cost |
| --- | --- |

## Resource Exchange

Resource exchange is bank-style conversion during the Spend Resources phase. It is not player trading or negotiation.

| Paid | Received |
| --- | --- |
| 1 Metal | 3 Wood |
| 1 Metal | 1 Stone and 1 Wood |
| 1 Stone | 2 Wood |

There is no trading up.

## Action Card Effects

| Card | Current Effect | Can Be Stopped By |
| --- | --- | --- |
| Raid Base | Attack another player's base using 1 unit. If not stopped, target base loses 1 point. | Counter or unit defense |
| Resource Theft | Steal 1 resource from another player. | Counter |
| Unit Kill | Destroy 1 unit controlled by another player. | Counter |
| Counter | Cancel an action card used against you or a Counter in the current response chain. | Counter |

## Supply Circulation

| Item | Current Rule |
| --- | --- |
| Bought units | Enter play immediately |
| Unit supply | Effectively unlimited |
| Resource supply | Effectively unlimited |
| Spent resources | Return to circulation |
| Discarded units | Return to circulation |
| Triangles sacrificed for action cards | Return to circulation |
| Sacrifice for action card | Triangle only |
| Used action cards | Move to action card discard pile after resolution |
| Empty action card deck | Shuffle discard pile into a new deck when a draw is needed |
| Empty deck and empty discard pile | No card is drawn |
| Raiding unit | Discarded after Raid Base resolution whether raid succeeds or fails |
| Defending raid units | Discarded after Raid Base resolution whether raid succeeds or fails |

## Game End

| Condition | Result |
| --- | --- |
| Player base reaches 0 points | That player is eliminated |
| Only one player remains | Game ends; remaining player wins |
